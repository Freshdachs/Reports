Implementing the Metropolis Iteration function¶

What actually differentiates the Metropolis algorithm from other variants is the actual iteration function. In this function, we determine a proposal and decide if we accept the new sample or keep the old one.

But for other algorithms, we could simply replace the iteration function.

In [111]:
# f: proportial to density, p: proposal, i: previous/initial value
def iteration(f,p,i):
        x = p(i)
        return x if f(x)/f(i) > np.random.uniform() else i
    
def sample(f,p,i,n):
    iteratee = lambda i: iteration(f,p,i)
    return islice(iterate(iteratee,i),0,n)

def l_egg(x,y):
    return np.exp((2+np.cos(x/2)*np.cos(y/2))**5)

#uniform prior 0..10*pi for x and y 
frm_egg = [0,0]
to_egg = [10*np.pi, 10*np.pi]

def prior_egg(*pt):
    return 1*(all(np.greater(pt,frm_egg)) and all(np.less(pt, np.array(to_egg))))

def l_p_egg(x,y):
    return prior_egg(x,y)*l_egg(x,y)

Metropolis Hastings¶

The Metropolis Hastings algorithm is a more general form of the Metropolis Algorithm. It changes the algorithm, so that it also works with asymmetric Jump Distributions. The only requirement is that densities for the jump distributions can be computed. During the sampling process, we no longer only look at the ratio of densities, but at the ratio of densities from the target distribution normalized with the densities we have from the jump distribution. So when comparing densities, we now take the density into account, with which the sample has been generated (density of the jump distribution).

The new ratio computes as follows: $$r = \frac{p(\theta^*|y)/J_t(\theta^*|\theta^{t-1})}{p(\theta^{t-1}|y)/J_t(\theta^{t-1}|\theta^*)}$$

# f: proportial to density, p: proposal, i: previous/initial value, 
# j: proportional to density of jump distribution, takes 2* the parameters
def iteration_mh(f,p,i,j):
        x = p(*i)
        return x if (f(*x)/j(*(x+i)) )/(f(*i)/j(*(i+x)) ) > np.random.uniform() else i
        
        
def sample_mh(f,p,i,n,j):
    for k in range(n):
        i=iteration_mh(f,p,i,j)
        yield i
